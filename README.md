# E36Clock
Radio controlled clock using a BMW E36 instrument cluster as Display.

# Wiring
| Pin Number | Socket color |      Function     | Arduino Pin |
|:----------:|:------------:|:-----------------:|:-----------:|
|      2     |     White    |       +12 V       |             |
|     19     |     White    |        GND        |             |
|     20     |     White    |        RPM        |      7      |
|     22     |     White    |        +12V       |             |
|     23     |     White    |       +12 V       |             |
|     24     |     White    |        GND        |             |
|      3     |     Blue     |        GND        |             |
|      4     |     Blue     |        +12V       |             |
|      5     |     Blue     |        +12V       |             |
|     16     |     Blue     |        +12V       |             |
|     18     |     Blue     | Water temperature |      6      |
|      1     |     Black    |        Fuel       |      5      |
|      3     |     Black    |        GND        |             |
|      4     |     Black    |       Speed       |      4      |
